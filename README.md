# RailSathi AI 🚂

An AI-Powered Railway Travel Assistant built for hackathons, utilizing **Next.js 15 App Router**, **Tailwind CSS**, and **Google Gemini API**.

## Features
- Generative AI responses strictly formatted in JSON.
- Responsive, modern Government/Railway-tech inspired UI.
- Pre-configured Framer Motion animations.
- Quick suggested prompts.

## Tech Stack
- Next.js 15 (React 19)
- TypeScript
- Tailwind CSS
- Google Gemini API (`@google/generative-ai`)
- Lucide React Icons
